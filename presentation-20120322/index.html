An HTML Primer

What is HTML?

  1. HyperText Markup Language
  2. Three major flavors: HTML, XHTML, HTML 5
  3. Made up of tags

“Markup Language”

  1. Write HTML for the meaning, not the display
  2. Easy for humans:
    Meet Miles for lunch, today at noon.
  3. Easy for robots:
    <event eventDateTime="2012-03-22 12:00:00">
	<persons>
		<person name="Miles" />
	</persons>
</event>
  4. Okay for both:
    <div class="event">Meet <a href="http://github.com/awayken" class="person">Miles</a> for lunch, <span class="datetime" title="2012-03-22 12:00:00">today at noon</span>.</div>

Anatomy of a tag

  1. Tag name
  2. Attribute name
  3. Attribute value
  4. Tag content or text

A simple webpage

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
	"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
	<title>Welcome to Webpage</title>
</head>
<body>
	<h1>Welcome!</h1>
	<p>Welcome to our website!</p>
</body>
</html>

<head>

  1. Metadata of web page
  2. Made for browsers and search engines
  3. Common tags: <link>, <meta>, <script>, <style>, <title>

<body>

  1. Content of web page
  2. Made for browsers, search engines, and humans
  3. Common tags: everything else

  2. Dress it up: bold, emphasis, blockquote
