An HTML Primer

What is HTML?

  • HyperText Markup Language
  • Made up of tags, which mark up your content
  • Tags adds meaning for humans and robots

Anatomy of a tag

  • <p>Hello!</p>
  • <a href="index.html">Homepage</a>
  • <img src="logo.jpg" height="100" />
  • Tag name – starts and ends tags
  • Attribute name – lowercase, equal after
  • Attribute value – surround in quotes, equal before
  • Tag content or text – must be "typeable", can be more HTML

Anatomy of a tag (quiz)

  • <div class="tree">Birch Tree</div>
  • <p>I <strong>really really <em>really</em></strong> disagree.</p>
  • <link rel="stylesheet" type="text/css" href="styles.css" />
  • <textarea id="mytext" name="mytext" cols="30" rows="5">Type here.</textarea>

A simple webpage

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
	"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
	<title>Welcome to Webpage</title>
</head>
<body>
	<h1>Welcome!</h1>
	<p>Welcome to our website!</p>
</body>
</html>
  • Doctype – important, hard to remember so copy & paste
  • <html> – wraps the whole page
  • <head>
  • <body>

<head>

  • Metadata of web page
  • Made for browsers and search engines
  • Common tags: <link>, <meta>, <script>, <style>, <title>

<body>

  • Content of web page
  • Made for browsers, search engines, and humans
  • Common tags: everything else
  • Common attributes: id, class, title, style

Block elements

  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6> – headings, decreasingly important
  • <p> – paragraphs
  • <blockquote> – blockquote – cite
  • <ul>, <ol> – lists, unordered and ordered
  • <li> – a list item (used for both kinds of lists)
  • <dl>, <dt>, <dd> – definition list, with definition term and definition data
  • <div> – division

Inline elements

  • <a> – anchor, creates a hyperlink – href, target
  • <em> – emphasis, usually adds italic
  • <strong> – strong emphasis, usually adds bold
  • <code> – computer code, usually uses monospaced font
  • <sup> – superscript
  • <sub> – subscript
  • <br /> – adds a line break (no closing tag)
  • <span> – like <div> but stays in line

Media and Frame elements

  • <img /> – images (no closing tag) – src, alt, width, height
  • <object>, <param />, <embed> – for embedding all kinds of crazy media – see Internet for examples
  • <iframe> – used to display another URL in your HTML – src, width, height, name, frameborder

Table elements

  • <table> – creates the table – width, summary
  • <thead>, <tbody>, <tfoot> – organizes your table into head, body, and foot sections
  • <tr> – table row
  • <th> – table heading cell – colspan, rowspan, valign
  • <td> – table data cell – colspan, rowspan, valign
