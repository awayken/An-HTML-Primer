An HTML Primer

What is HTML?

  • HyperText Markup Language
  • Three major flavors: HTML, XHTML, HTML 5
  • Made up of tags

“Markup Language”

  1. Easy for humans:
    Meet Miles for lunch, today at noon.
  2. Easy for robots:
    <event eventDateTime="2012-03-22 12:00:00">
	<persons>
		<person name="Miles" />
	</persons>
</event>
  3. Okay for both:
    <div class="event">Meet <a href="http://github.com/awayken" class="person">Miles</a> for lunch, <span class="datetime" title="2012-03-22 12:00:00">today at noon</span>.</div>
  • Write HTML for the meaning, not the display

Anatomy of a tag

  • <p>Hello!</p>
  • <a href="index.html">Homepage</a>
  • <img src="logo.jpg" height="100" />
  • Tag name – starts and ends tags
  • Attribute name – lowercase, equal after
  • Attribute value – surround in quotes, equal before
  • Tag content or text – must be "typeable", can be more HTML

Anatomy of a tag (quiz)

  • <div class="tree">Birch Tree</div>
  • <p>I <strong>really really <em>really</em></strong> disagree.</p>
  • <link rel="stylesheet" type="text/css" href="styles.css" />
  • <textarea id="mytext" name="mytext" cols="30" rows="5">Type here.</textarea>

A simple webpage

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
	"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
	<title>Welcome to Webpage</title>
</head>
<body>
	<h1>Welcome!</h1>
	<p>Welcome to our website!</p>
</body>
</html>
  • Doctype – important, hard to remember so copy & paste
  • <html> – wraps the whole page
  • <head>
  • <body>

<head>

  • Metadata of web page
  • Made for browsers and search engines
  • Common tags: <link>, <meta>, <script>, <style>, <title>

<body>

  • Content of web page
  • Made for browsers, search engines, and humans
  • Common tags: everything else

Block elements

  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6> – headings, decreasingly important
  • <p> – paragraphs
  • <ul>, <ol> – lists, unordered and ordered
  • <li> – a list item (used for both kinds of lists)
  • <div> – division

Inline elements

  • <a> – anchor, creates a hyperlink
  • <em> – emphasis, usually adds italic
  • <strong> – strength, usually adds bold
  • <sup> – superscript
  • <sub> – subscript
  • <br /> – adds a line break (no closing tag)
  • <span> – like <div> but stays in line

Media elements

  • <img> – images (no closing tag)
  • <object>, <embed>, <iframe> – for embedding all kinds of crazy media (see Internet for examples)

Table elements

  • <table> – creates the table
  • <thead>, <tbody> – organizes your table into head and body sections
  • <tr> – table row
  • <th> – table heading cell
  • <td> – table data cell

<html xmlns="http://www.w3.org/1999/xhtml">
<head>
	<title>Let’s write some HTML!</title>
</head>
<body>
</body>
</html>
  1. Create a paragraph (<p>): I crawled out almost immediately, and crouched, my feet still in the water, under a clump of furze.
  2. Add strong emphasis (<strong>) to the words: "almost immediately"
  3. Hyperlink (<a>) the last word, furze, to: http://en.wiktionary.org/wiki/furze

Let’s write some HTML!

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
	"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

</body>
</html>
